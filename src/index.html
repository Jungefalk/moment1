Om uppgiften

Uppgiften går ut på att, genom att skapa en boilerplate, förstå hur man kan använda Node js för att förenkla arbetsprocessen vid utvecklingen av en webbplats. Att förstå att man med hjälp av npm och parcel kan använda sig av verktyg för att automatisera funktioner på webbplatsen som exempelvis optimera grafik. Men också att lära sig hur man kan automatisera sin utvecklingsmiljö för att enkelt uppdatera en redan publicerad webbplats samt automatiskt minifiera html, css och javaScript för att optimera filerna.
